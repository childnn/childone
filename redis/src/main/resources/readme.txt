主从配置:
1. 主写从读
2. 启动多个端口 redis, 默认都是 master
   比如启动 6379/6380/6381 三个端口实例
   命令配置, 进入 6380/6381, 执行命令 slaveof 127.0.0.1 6379   # 当前实例为 6379 的从实例
   另, 可以通过修改 redis.conf 配置文件: 查找 replication
3. 查看主从信息: info replication
4. 如果主机断开, 可以在从机执行命令 slaveof no one 将该从机升级为 master
5. 上述通过命令修改主机, 哨兵模式自动选择主机
   三个哨兵, 一主二从, 至少 6 个进程
   redis 本身是集群, 哨兵也是集群
