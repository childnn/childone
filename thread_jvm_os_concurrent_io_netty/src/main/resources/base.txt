关于 printf 函数的占位符：
  %d    按照十进制整数打印
  %6d   按照十进制整数打印，至少 6 个字符宽
  %f    按照浮点数打印
  %6f   按照浮点数打印，至少 6 个字符宽
  %.2f  按照浮点数打印,小数点后有两位小数
  %6.2f 按照浮点数打印,至少 6 个字符宽, 小数点后有两位小数
  %o    八进制数
  %x    十六进制数
  %c    字符
  %s    字符串
  %%    百分号本身
  \t    横向制表符
  \v    纵向制表符

C 语言:
#define 指令  定义常量
#define 常量名 替换文本(可以是数字或字符串)

C-code: 把输入一次一个字符地复制到输出
    读一个字符
    while(该字符不是文件结束指示符){
        输出刚读入的字符
        读下一个字符
    }
---------------------------------------------
#include <stdio.h>
main() {
    int c; // 之所以把 c 声明成 int 类型而不是 char 类型,必须让它大到除了存储任何可能的字符外还要能存储文件结束符 EOF.
    c = getchar();  // 标准库提供,从文本流中读入下一个输入的字符,并将其作为结果返回.
    while(c != EOF) { // end of file, 文件结束
        putchar(c);
        c = getchar(); // 标准库提供,打印一个字符.
    }
    ==> 简化
    while((c = getchar()) != EOF)
        putchar(c);
}

define：定义，表示创建或分配存储单元；
declaration：声明，说明变量的性质，但并不分配存储单元。
